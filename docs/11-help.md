# F10 · 도움말 (앱 내 매뉴얼)

## 목적
날짜 표기·반복 키워드·점 클릭처럼 **외워야 쓰는 기능들을 앱 안에서 바로 찾아보게** 한다.
기능이 늘 때마다 어차피 잊어버리므로, 매뉴얼은 앱 안에 있어야 한다.

## 요구사항 (검증 항목)
- [ ] R1. ⋯ 메뉴에 **"도움말"** — 누르면 매뉴얼 화면이 열리고 "← 뒤로"로 복귀 (휴지통과 같은 방식).
- [ ] R2. 내용은 **실제 동작과 일치**해야 한다 (기능이 바뀌면 도움말도 갱신 — spec-check 대상):
      빠른 입력(날짜 표기·반복 키워드·📅 버튼), 우선순위 점, 완료/실행취소,
      탭 안내, 알아둘 것(공지/참고), 백업(내보내기/가져오기/휴지통), 데이터 저장 위치.
- [ ] R3. 한 화면 분량으로 간단하게 — 표기 예시는 코드체로 눈에 띄게.

## 관련 파일 (구현하면서 채움)
- (예정) `src/components/HelpView.jsx`, `src/App.jsx` (⋯ 메뉴 연결)

## 비고
- 도움말 내용의 원본은 이 코드(HelpView.jsx) 하나 — 별도 문서와 이중 관리하지 않는다.
