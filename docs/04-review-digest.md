# F04 · 정리본(리뷰) 화면

> 🔜 **스텁 문서**이며, 형태 자체가 아직 **미확정**입니다.
> 사용자가 마일스톤 1~2를 실제로 써본 뒤에 무엇이 필요한지 정하고 상세화합니다.

## 목적 (초안)
쌓인 할 일을 한 발 떨어져서 훑어보는 화면. "정리본"이 무엇인지는 써보며 결정.

## 후보 아이디어 (착수 시 선택)
- (a) 주간 돌아보기: 이번 주 완료한 것 / 남은 것 요약
- (b) 방치된 할 일 모아보기: 오래 안 건드린 항목 정리(완료/삭제/날짜 재지정) 유도
- (c) 하루 시작 화면: 오늘 할 일 + 밀린 일 요약

## 미결정
- 위 후보 중 무엇이 실제로 필요한지 — **M1~2 사용 경험 후 결정.**
