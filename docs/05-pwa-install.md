# F05 · PWA 설치 (선택)

> 🔜 **스텁 문서**입니다. 할지 말지 자체가 선택 사항.

## 목적 (초안)
브라우저 주소를 치는 대신, 바탕화면 아이콘으로 여는 **설치된 프로그램처럼** 만들기.
(manifest + service worker. 컴퓨터 전용이므로 폰 설치는 고려 안 함)

## 요구사항 (착수 시 확정)
- [ ] R1. 웹 앱 매니페스트 (`name`, `icons`, `start_url`, `display`)
- [ ] R2. 서비스 워커로 정적 파일 캐시 → 오프라인에서도 열림
- [ ] R3. 설치 후 독립 창으로 실행됨

## 미결정
- `vite-plugin-pwa` 사용 여부 → 착수 시 결정.
