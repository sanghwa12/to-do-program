# F07 · 달력 뷰 (월간)

## 목적
날짜가 있는 할 일(마감·당일·기간)을 **월간 달력 위에서 직관적으로** 본다.
목록으로 읽는 "날짜" 탭과 달리, 한 달의 그림이 한눈에 들어오는 화면.

## 사용자 시나리오
- "달력" 탭을 누르면 이번 달 달력이 뜨고, 각 날짜 칸에 그날의 할 일 제목이 보인다.
- "학부실험 조교 11/11~11/23" 같은 기간 일은 그 기간의 **모든 칸**에 걸쳐 보인다.
- 날짜 칸을 클릭하면 아래에 그 날짜의 할 일 목록이 펼쳐지고, 거기서 바로 완료/수정/삭제한다.
- ◀ ▶로 이전·다음 달을 넘겨보고, "오늘" 버튼으로 바로 돌아온다.

## 요구사항 (검증 항목)
- [ ] R1. "달력" 탭이 있고, 월간 격자(일~토, 7열)를 보여준다. 처음 열면 **이번 달**.
- [ ] R2. ◀/▶ 버튼으로 이전·다음 달 이동(해 넘김 포함), "오늘" 버튼으로 이번 달로 복귀.
- [ ] R3. 날짜 있는 할 일이 해당 칸에 **제목 칩**으로 표시된다:
      - 마감(due)·당일(day) → `dueDate` 칸에
      - 기간(range) → 시작일~끝일의 **모든 칸**에
- [ ] R4. **오늘 칸**이 시각적으로 강조된다.
- [ ] R5. 한 칸에 칩은 최대 3개까지, 넘치면 "+N"으로 표시한다.
- [ ] R6. 날짜 칸을 클릭하면 달력 아래에 그 날짜의 할 일 목록이 나타나고,
      완료/수정/삭제가 다른 탭과 동일하게 동작한다. 같은 칸을 다시 클릭하면 접힌다.
- [ ] R7. 완료된 할 일의 칩은 흐리게+취소선으로 구분된다.
      (D00 색 원칙 준수: 달력에 새 강조색을 추가하지 않는다)
- [ ] R8. 외부 달력 라이브러리 없이 구현한다.
      (이유: 월 격자는 단순한 계산이라 직접 만드는 게 용량·의존성 면에서 낫고, 코드 공부에도 좋음)

## 관련 제약 (CLAUDE.md / D00)
- 저장된 데이터를 읽어 화면에서만 계산 (서버·외부 서비스 ❌).
- 색 최소화: 강조는 기존 파랑(오늘/선택) 재사용, 빨강 추가 금지.
- 휴지통에 있는 할 일은 달력에 안 보인다 (목록과 동일 규칙).
