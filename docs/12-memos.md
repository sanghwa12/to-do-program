# F11 · 메모장 (자유 글)

## 목적
할 일도, 한 줄 참고도 아닌 **여러 줄 자유 글**을 담는 곳.
(회의 기록, 초안, 생각 정리 — 예전 윈도우 스티커 메모에서 하던 것들의 최종 이사처)

## 다른 보관함과의 구분
| | 할 일 | 📎 참고 (알아둘 것) | 📝 메모 |
|---|---|---|---|
| 형태 | 한 줄 + 체크 | 제목 + 설명 한 줄 | **여러 줄 자유 글** |
| 예 | 시약 주문 | 외부 강연 → 윤리센터 | 회의 기록, 편지 초안 |

## 데이터 모델
```ts
type Memo = {
  id: string;        // 자동 생성
  text: string;      // 자유 글 (여러 줄). 첫 줄이 목록에서 제목 역할
  createdAt: string; // 자동
  updatedAt: string; // 수정할 때마다 갱신 — 목록 정렬 기준
};
```
